# 📅 Sobre
Testes automatizados na aplicação Cypress Real World, utilizada para aprender e praticar testes automatizados no Cypress.

# 🌐 Tecnologias usadas
- Cypress 

- Javascript 

# </> Como rodar a aplicação Cypress Real World
1. Clone o repositório: ``git clone https://github.com/cypress-io/cypress-realworld-app.git``
2. Depois de clonado, mude para o diretório cypress-realworld-app e instale todas as dependências npm usando Yarn. ``yarn install``
3. Para rodar a aplicação utilize ``yarn dev``
4. Assim que a aplicação estiver instalada, abra uma janela no browser do seu navegador utilizando a seguinte URL: http://localhost:3000
